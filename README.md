## 项目说明
本博客用于记录个人学习笔记、项目开发记录与技术实践总结，主要内容方向包括：
- 计算机专业课程学习笔记
- 项目开发过程与经验分享
- 技术问题解决记录与实践心得

---

## 技术栈
- 前端：HTML / CSS / JavaScript
- 部署：GitHub Pages 静态站点托管

---

## 仓库目录结构
- `2026/05/01/app-story/`：博客文章目录（按日期归档）
- `archives/`：文章归档页面
- `css/`：样式文件目录
- `fancybox/`：图片灯箱组件资源
- `js/`：脚本文件目录
- `index.html`：博客首页
- `README.md`：项目说明文档（即本文）

## 功能说明
- 响应式布局，适配多端设备阅读
- 按日期归档文章，便于内容回顾与检索
- 支持图片灯箱查看效果
